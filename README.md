# Resumo do trabalho
Este trabalho investiga a aplicação de **GraphRAG** (*Graph Retrieval-Augmented Generation*) como estratégia para exploração, organização e consolidação de conhecimento sobre métodos de **Fine-Tuning (FT)** em LLMs. A partir de um acervo de documentos (artigos e sites específicos), o conteúdo foi injetado na ferramenta **EdgeQuake** e **Graphify** para construção de um grafo de conhecimento com milhares de entidades e relacionamentos identificáveis, permitindo uma visão estruturada das principais técnicas bem como seus requisitos, vantagens e suas limitações. O estudo de caso evidenciou que a combinação de GraphRAG com LLMs facilitou a parametrização e comparação de técnicas de FT em situação de aplicações com diferentes naturezas de dados e/ou tarefas.
